## Was WORTLIGA nicht kann – und das sollte man wissen

Ehrlich gesagt: WORTLIGA ersetzt keinen Lektor. Das sagen auch die Nutzer selbst.

Das Tool ist schematisch. Es meldet Füllwörter, auch wenn sie bewusst gesetzt sind. Es kritisiert lange Sätze, auch wenn der Rhythmus gewollt ist. Ein Autor mit gefestigtem Stilgefühl wird manchen Hinweis wegklicken.

Die Rechtschreibprüfung deckt nicht alles ab – bei hochwertigen, komplexen Texten bleibt das Korrekturlesen unverzichtbar. Und die kostenlose Version mit 1.500 Zeichen ist für ernsthafte Nutzung wirklich zu begrenzt.

Heißt: WORTLIGA ist ein starkes Hilfsmittel. Kein Ersatz für Schreibtalent.
